# 品牌演进记录 (Branding Evolution)

## 演进历程

### 1. 初始阶段: PixelDesk
*   **含义**: 像素工位的直译。
*   **状态**: 弃用。
*   **原因**: 名字较为大众化，缺乏独特性，且中文名已确定为“象素工坊”，直译英文略显生硬。

### 2. 过渡阶段: Elephixel Workshop
*   **含义**: Elephant (大象) + Pixel (像素) 的组合。
*   **状态**: 弃用。
*   **原因**:
    *   拼写过于复杂（用户反馈难以拼写）。
    *   域名 `elepixel.com` 已被注册。
    *   品牌识别度较低。

### 3. 定型阶段: Tembo PX Workshop
*   **拼写**: **T-E-M-B-O  P-X**
*   **核心含义**:
    *   **Tembo**: 斯瓦希里语 (Swahili) 中的“大象”。代表智慧、力量、长寿与可靠。
    *   **PX**: 全球技术与设计领域“像素 (Pixel)”的通用缩写。
    *   **Workshop**: 延续“工坊”的概念，强调匠心创作。
*   **优势**:
    *   **极简易读**: Tembo 为 CVC 结构（辅音-元音-辅音），PX 只有两个字母，几乎不会拼错。
    *   **独特且高级**: 避开了常见的合成词，带有异域文化底蕴，国际化视野更强。
    *   **域名友好**: 作为一个独特的组合，更利于注册域名及社交媒体账号。
    *   **高度对仗**: 完美呼应中文名“象(**Tembo**)素(**PX**)工坊”。

## 品牌理念
**Tembo PX Workshop** 致力于打造一个稳重如象、精细如像素的数字社交办公平台。通过斯瓦希里语的古老底蕴与 PX 的现代科技感结合，不仅是一个办公工具，更是一个充满智慧与连接的创作工坊。
